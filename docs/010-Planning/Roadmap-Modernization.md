# Roadmap: ModernFlyouts Fluent Design 2 Modernization

## Vision

Transform ModernFlyouts into a world-class, premium UI utility that feels native to Windows 11 and paves the way for the future of Windows personalization.

## Phase 1: Foundation (Current)

- [x] Deep Research into Fluent Design 2 and iNKORE library.
- [ ] Dependency Migration: Replace `ModernWpfUI` with `iNKORE.UI.WPF.Modern`.
- [ ] Initialize Fluent 2 Theme & Typography (Segoe UI Variable).
- [ ] Basic Mica backdrop implementation for all flyouts.

## Phase 2: Refinement

- [ ] Implement Mica Alt for active flyouts.
- [ ] Fine-tune corner radii (8px/4px) across all controls.
- [ ] Update all icons to Fluent System Icons (SegMDL2 to Fluent Icons).
- [ ] Introduce Spring Velocity animations for flyout transitions.

## Phase 3: Premium Experience

- [ ] Implement "Connected Animations" for media session transitions.
- [ ] Desktop Acrylic support for secondary windows.
- [ ] Smart Accent Color integration.
- [ ] Custom JSON-based Theme Support.

## Phase 4: Verification & Release

- [ ] Comprehensive testing on Win10 and Win11.
- [ ] Performance optimization for backdrop effects.
- [ ] Final release notes and community walkthrough.
